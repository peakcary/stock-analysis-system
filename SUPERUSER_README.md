## 🔧 技术说明

### 权限实现
- 前端通过检查 `user.memberType === 'premium'` 来提供超级管理员权限
- 后端通过检查会员类型和查询次数来控制访问权限
- 查询次数设置为999,999，实际上等同于无限制

### 安全建议
1. **修改默认密码**: 建议在首次登录后立即修改密码
2. **保护账户信息**: 请妥善保管登录凭据，不要与他人分享
3. **定期检查**: 定期检查账户活动和数据访问情况
