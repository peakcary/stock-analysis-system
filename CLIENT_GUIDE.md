## 🔧 分别启动服务

### 方式1: 完整启动 (推荐)
```bash
./start_all_services.sh
```

### 方式2: 单独启动
```bash
# 只启动后端
cd backend && source venv/bin/activate && uvicorn app.main:app --port 8000 --reload

# 只启动管理后台
cd frontend && npm start

# 只启动客户端
cd client && npm run dev
```

### 方式3: 使用现有脚本
```bash
./quick_start.sh        # 启动后端 + 管理后台
./start_client.sh       # 单独启动客户端
```
